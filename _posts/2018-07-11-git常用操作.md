`git add -A`: 将改动添加到暂存区                                                        
`git commit -m ***`: 将暂存区内容提交到本地仓库并附加说明信息                                                 
`git log`: 显示从近到远的提交日志(参数--pretty=oneline可以精简显示信息)                                    
`git reset --hard HEAD^`: 用于回退本地仓库中的版本, HEAD表示当前版本, HEAD^就是前一个版本, 前10个版本可以写成HEAD~10
