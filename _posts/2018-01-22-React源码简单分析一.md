源码版本为16.0.0-alpha.3

React受欢迎的一个原因是可以自定义组件,这样不但可以复用开发好的组件,也可以使代码结构更清晰。

ES5创建组件可以使用`React.createClass()`,ES6中使用`class myComponent extends React.Component`,其实内部还是调用的`React.createClass()`。

下面我们简单分析一下一个组件从创建到渲染经历了哪些过程(我的理解。。)

使用ES6写一个组件
```
class HelloWorld extends React.Component{
  constructor(props){
    super(props);
    this.state = {
      state: 'state'
    }
  }
  
  componentWillMount(){
    console.log('componentWillMount')
  }
  
  componentDidMount(){
    console.log('componentDidMount')
  }
  
  render(){
    return(
      <h1>
        Hello World
      </h1>
    )
  }
}

HelloWorld.defaultProps = {
  props: 'props'
}
```

ES5写法
```
const HelloWorld = React.createClass({
  getDefaultProps: function(){
    return{
      props: 'props'
    }
  },
  
  getInitialState: function(){
    return{
      state: 'state'
    }
  },
  
  componentWillMount: function(){
    console.log('componentWillMount')
  },
  
  componentDidMount: function(){
    console.log('componentDidMount')
  },
  
  render: function(){
    return(
      <h1>
        Hello World
      </h1>
    )
  }
})
```

执行渲染函数就会在页面中渲染
```
ReactDOM.render(
  <HelloWorld />,
  document.getElementById('root')
)
```
解析后会变成
```
ReactDOM.render(
  React.createElement(HelloWorld),
  document.getElementById('root')
)
```

首先分析一下`createClass()`源码(src/isomorphic/classic/class/ReactClass.js)
省略了一些错误提示代码

```
var ReactClass = {

  /**
   * Creates a composite component class given a class specification.
   *
   * @param {object} spec Class specification (which must define `render`).
   * @return {function} Component constructor function.
   * @public
   */
  createClass: function(spec) {    
    var Constructor = function(props, context, updater) {
      if (this.__reactAutoBindPairs.length) {
        bindAutoBindMethods(this);
      }

      this.props = props;
      this.context = context;
      this.refs = emptyObject;
      this.updater = updater || ReactNoopUpdateQueue;

      this.state = null;

      // ReactClasses doesn't have constructors. Instead, they use the
      // getInitialState and componentWillMount methods for initialization.

      var initialState = this.getInitialState ? this.getInitialState() : null;

      this.state = initialState;
    };
    Constructor.prototype = new ReactClassComponent();
    Constructor.prototype.constructor = Constructor;
    Constructor.prototype.__reactAutoBindPairs = [];
    
    //将spec中的属性绑定到Constructor的prototype
    mixSpecIntoComponent(Constructor, spec);

    // Initialize the defaultProps property after all mixins have been merged.
    if (Constructor.getDefaultProps) {
      Constructor.defaultProps = Constructor.getDefaultProps();
    }

    // Reduce time spent doing lookups by setting these on the prototype.
    for (var methodName in ReactClassInterface) {
      if (!Constructor.prototype[methodName]) {
        Constructor.prototype[methodName] = null;
      }
    }

    return Constructor;
  }
};
```

`createClass()`主要干了几件事:
1. 定义构造方法`Constructor`,原型为`ReactClassComponent`的对象
2. 进行`props`、`refs`等的初始化,调用`getInitialState()`来初始化`state`并调用`getDefaultProps()`来初始化静态变量`defaultProps`
3. 返回构造函数`Constructor`

`createElement()`源码(src/isomorphic/classic/element/ReactElement.js)

```
ReactElement.createElement = function(type, config, children) {
  var propName;

  // Reserved names are extracted
  var props = {};

  var key = null;
  var ref = null;
  var self = null;
  var source = null;

  //从config中提取内容,并赋值给ref,key,props
  if (config != null) {
    if (hasValidRef(config)) {
      ref = config.ref;
    }
    if (hasValidKey(config)) {
      key = '' + config.key;
    }

    self = config.__self === undefined ? null : config.__self;
    source = config.__source === undefined ? null : config.__source;
    // Remaining properties are added to a new props object
    for (propName in config) {
      if (hasOwnProperty.call(config, propName) &&
          !RESERVED_PROPS.hasOwnProperty(propName)) {
        props[propName] = config[propName];
      }
    }
  }

  // Children can be more than one argument, and those are transferred onto
  // the newly allocated props object.
  //从参数的第三个开始都是children参数,挂到`props.children`属性上
  var childrenLength = arguments.length - 2;
  if (childrenLength === 1) {
    props.children = children;
  } else if (childrenLength > 1) {
    var childArray = Array(childrenLength);
    for (var i = 0; i < childrenLength; i++) {
      childArray[i] = arguments[i + 2];
    }
    props.children = childArray;
  }

  // Resolve default props
  //取出静态变量并给并给未设置值的属性设置默认值
  if (type && type.defaultProps) {
    var defaultProps = type.defaultProps;
    for (propName in defaultProps) {
      if (props[propName] === undefined) {
        props[propName] = defaultProps[propName];
      }
    }
  }
  
  //返回一个ReactElement对象
  return ReactElement(
    type,
    key,
    ref,
    self,
    source,
    ReactCurrentOwner.current,
    props
  );
};
```
