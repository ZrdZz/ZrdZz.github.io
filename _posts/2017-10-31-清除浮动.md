1. 给父元素添加属性`overflow: hidden`
   缺点： 元素区域以外的部分会被隐藏
2. 使用额外的标签： 在浮动的盒子下面放一个标签，然后使用`clear: both`
   缺点： 会增加无用的标签，造成结构混乱
3. 使用伪元素清除浮动
   ```
   .clearfix:after{
     content: "";
     height: 0;
     display: block;    //默认为inline，clear只能应用于block元素
     visibility: hidden;
     clear: both;
   }
   ```
## 最好使用第三种方法
